description: Применять ко всем языкам программирования при
  генерации/редактировании кода для русскоязычных пользователей. Только чистый
  исполняемый код без комментариев.
alwaysApply: true
---

Никогда не добавляй комментарии в код (ни //, ни #, ни /* */). Код должен быть чистым без комментариев. Комментарии пиши только в текстовом объяснении отдельно от кода.
