ИСПОЛЬЗУЙ tools для редакторивания, создания файлов. Если какая то ошибка в редактировании файла то прочти его сначала. Если ошибка в создании файла убедись что ты в нужном месте создаешь файл. Для удаления фала используй команду в терминале. Для перемещения файла используй команду в терминале, при этом не забудь что нужно зависимости обновить, это можно делать через поиск.
Никогда не добавляй комментарии в код. Код должен быть чистым без комментариев. Комментарии пиши только в текстовом объяснении отдельно от кода. Джавадок, докстринг итп писать можно если пользователь явно это скзаал.
